# 🤖 Robotics & AI Control (C++)
This repository serves as the central hub for my robotics journey. From classic hardware-control algorithms to AI-driven environment analysis, I am exploring how software interacts with the physical world.

## 🚀 Projects & Progress
- **[Autonomous Navigation]:** Implementing Obstacle Avoidance and Line Following logic using IR and Ultrasonic sensors.
- **[AI Vision]:** Developing a computer vision system to analyze environments in real-time through the robot's camera "eyes."

## 🛠️ Tech Stack & Learning Goals
- **Languages:** C++ (Arduino/Embedded), Python (AI & Computer Vision).
- **Core Hardware:** Arduino, Motor Drivers, Ultrasonic Sensors, Cameras.
- **AI Integration:** Exploring OpenCV and basic Neural Networks for object detection.

## 🧠 Logic & Algorithms
1. **Low-Level Control:** Using C++ for real-time sensor polling and motor PWM (Pulse Width Modulation) adjustments.
2. **Environment Analysis:** Implementing conditional branching for obstacle redirection.
3. **Computer Vision (In Progress):** Applying logic to process camera frames and identify environmental patterns.
